# Descricion
Backend que distribuye los gastos de un grupo

# Programas pre instalados
 - docker
 - nodejs

 # Instalacion de dependencias
 ```
 npm install
 ```

 # Levantar depdencias con docker
 ```
 docker compose up
 ```

 # Levantar proyecto
 ```
 npm run start:dev
 ```
